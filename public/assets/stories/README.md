# 🎬 스토리(하이라이트) 영상 넣는 곳

인스타 하이라이트를 **화면 녹화한 영상(mp4)**을 여기에 넣습니다.
(사진과 달리 영상은 최적화 대상이 아니라 이 public 폴더에 두고 경로로 참조해요.)

## 넣는 법
1. 인스타 하이라이트 1개를 처음부터 끝까지 **화면 녹화** (스티커·문구 그대로 재생되게)
2. mp4를 이 폴더에 넣기 — 이름은 영문 추천 (예: `serengeti.mp4`)
3. "넣었어 / 파일명 ○○" 알려주면 → 구간 시작시간을 자동 추출해서
   `src/data/stories.ts` 의 해당 reel에 `video` + segment별 `t` 를 채웁니다.

## 참고
- 용량이 크면(수백 MB+) GitHub엔 부적합 → 나중에 CDN(Cloudflare Stream 등)으로 이전.
  지금은 **테스트용 1개**만 넣어 흐름을 검증합니다.
- 세로(9:16) 영상이 가장 잘 맞아요 (스토리 비율).
