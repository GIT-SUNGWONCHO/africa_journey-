탄자니아·잔지바르 기념품은 **첫 제시가가 적정가의 3~4배**. 깎는 걸 전제로 부르니 흥정은 필수.

**요령**
- 첫 제시가의 **40~50%부터** 카운터 (예: $20 부르면 "$8" 던지기)
- **3~4곳 비교** 후 구매. 안 맞으면 **돌아서기** → 다시 부르는 경우 많음
- 스톤타운 **다라자니 시장**이 관광상점보다 30~50% 저렴
- 현금 USD 소액권 준비하면 흥정이 빠름

**품목별 적정가 (참고)**
| 품목 | 적정 타결가 |
|---|---|
| 향신료(정향·계피·카다멈) 100g | $0.8~2 |
| 목각(잔지바르 문·동물 피규어) | $4~20 |
| 캉가/키텐게 천 | $2~4 |
| 팅가팅가 그림(소품) | $5~10 |
| 탄자니아 커피 250g | $6~8 |

인기: 향신료, 캉가/키텐게 천, 팅가팅가 그림, 잔지바르 문 목각, 커피.

> 실제 산 가격 기억나는 거 있으면 "이건 이 가격에 샀다"로 바꿔 넣으면 더 생생해져요.
