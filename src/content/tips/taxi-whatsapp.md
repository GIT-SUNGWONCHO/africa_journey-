탄자니아·잔지바르는 **왓츠앱으로 기사와 직접 소통**하는 게 기본. 현지 기사들은 일정 조율에 매우 능숙하고 유연하다.

**예약 흐름**
1. 원하는 날짜와 전체 구간을 공유 → 총액/구간별 견적 받기
2. 첫날 공항 만남 장소·피켓(이름판) 여부 확인
3. 최종 일정표 보내고 **"CONFIRMED"** 답장 받아 확정
4. 세부 픽업 시간은 여행하면서 그때그때 조율(arrange as we go)
5. 다른 기사로 정하면, 견적 줬던 기사에게 **확실히 거절 메시지**를 보내둬야 공항에서 동선 안 꼬임

**실제 구간 요금 (우리가 딜한 5일 패키지 $200 구성)**
| 구간 | 요금 |
|---|---|
| 공항 → 스톤타운 (첫날 픽업) | $10 |
| 스톤타운 → 더 모라(능귀) | $30 |
| 파제 전일 투어 (왕복) | $80 |
| 더 모라 → 켄드와 왕복 (09–16시) | $50 *(취소 시 정산 제외)* |
| 더 모라 → 공항 (샌딩) | $30 |

→ 여러 구간을 **묶어서 5일 $200**으로 딜. **선불 완납 X, 매일 운행 끝나고 그날 금액만 정산**.

**알아둘 점**
- 픽업 시간 변경(10시→11시), 두고 온 물건 때문에 이전 숙소 경유 등 **돌발 요청도 왓츠앱으로 바로 조율** 가능.
- 더 락 같은 해안 명소는 파도로 퇴장이 지연될 수 있음 → "15~20분 뒤 나갈게요, 출발할 때 다시 연락"이라고 미리 공유하면 기사가 무작정 안 기다림.
- 매일 정산 문화라 **건강·날씨로 일정 취소 시 정산이 깔끔**("허리가 아파서 18일 승마는 취소"처럼 직관적 소통).

> 실전 왓츠앱 문구
> - 확정: *"Please check and reply with 'CONFIRMED'."*
> - 유연 조율: *"Can we arrange the exact pickup times for the other days later as we go?"*
> - 지연 공유: *"For safety reasons we can leave in 15–20 min. I'll text you as soon as we start moving!"*

※ 기사 왓츠앱 번호는 카카오 오픈채팅방 등에서 이미 다녀온 사람에게 받는 게 가장 확실. (다녀온 사람이 협상한 가격 그대로 진행 가능)
