# 🖼️ 히어로 메인 사진 넣는 곳

이 폴더에 사진 파일을 **그냥 넣기만** 하면 맨 위 'TANZANIA' 배경 슬라이드쇼에 자동으로 올라갑니다.

- 지원: `.jpg .jpeg .png .webp .avif`
- **리사이즈 불필요** — 원본 그대로 넣어도 Astro가 자동으로 웹용 최적화/리사이즈해요.
- 순서는 **파일명 정렬순**. 원하는 순서대로 보이게 하려면 `01.jpg`, `02.jpg` … 처럼 번호를 붙이세요.
- 추천: 가로로 긴(landscape) 임팩트 있는 컷 3~6장. 세렝게티·잔지바르 섞으면 좋아요.

사진을 넣고 브라우저를 새로고침하면 바로 반영됩니다. (파일이 하나도 없으면 색 그라데이션 자리표시자가 대신 표시돼요.)
