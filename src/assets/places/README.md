# 🏞️ 지도 장소 사진 넣는 곳

여기에 **파일명을 장소 id로** 맞춰 사진을 넣으면, 지도에서 그 핀을 눌렀을 때 상세 카드에 자동으로 들어갑니다. (Astro가 자동 최적화)

## 파일명 (= 장소 id)
- `serengeti.jpg` — 세렝게티
- `ngorongoro.jpg` — 응고롱고로
- `arusha.jpg` — 아루샤
- `nungwi.jpg` — 능귀
- `stonetown.jpg` — 스톤타운
- `paje.jpg` — 파제

(`.jpg .jpeg .png .webp` 가능. 한 장씩만 있으면 됨.)

- 카드 사진 영역은 **가로 4:3**이라 가로 사진이 잘 맞아요.
- 넣고 새로고침하면 바로 반영. 없으면 기존 그라데이션+아이콘 표시.
